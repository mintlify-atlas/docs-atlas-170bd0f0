## Get Live TV Info

Retrieves information about available live TV services.

<CodeGroup>
```bash cURL
curl -X GET "https://your-server/LiveTv/Info" \
  -H "Authorization: MediaBrowser Token=YOUR_API_KEY"
```
</CodeGroup>

### Endpoint

<ParamField path="method" type="string" required>
  GET
</ParamField>

<ParamField path="endpoint" type="string" required>
  `/LiveTv/Info`
</ParamField>

### Authentication

Requires `LiveTvAccess` policy authorization.

### Response

<ResponseField name="LiveTvInfo" type="object">
  Live TV service information
  <Expandable title="Properties">
    <ResponseField name="Services" type="array">
      Available live TV services
    </ResponseField>
    <ResponseField name="IsEnabled" type="boolean">
      Whether live TV is enabled
    </ResponseField>
    <ResponseField name="EnabledUsers" type="array">
      Users with live TV access
    </ResponseField>
  </Expandable>
</ResponseField>

### Status Codes

- `200` - Available live TV services returned successfully
- `401` - Unauthorized
- `403` - Forbidden

## Get Programs (GET)

Retrieves live TV program guide information (EPG data) using query parameters.

<CodeGroup>
```bash cURL
curl -X GET "https://your-server/LiveTv/Programs?channelIds=CHANNEL_ID&limit=50" \
  -H "Authorization: MediaBrowser Token=YOUR_API_KEY"
```
</CodeGroup>

### Endpoint

<ParamField path="method" type="string" required>
  GET
</ParamField>

<ParamField path="endpoint" type="string" required>
  `/LiveTv/Programs`
</ParamField>

### Authentication

Requires `LiveTvAccess` policy authorization.

### Query Parameters

<ParamField query="channelIds" type="array" required>
  The channels to return guide information for (comma-delimited GUIDs)
</ParamField>

<ParamField query="userId" type="string (Guid)">
  Filter by user ID
</ParamField>

<ParamField query="minStartDate" type="string (DateTime)">
  The minimum premiere start date
</ParamField>

<ParamField query="hasAired" type="boolean">
  Filter by programs that have completed airing, or not
</ParamField>

<ParamField query="isAiring" type="boolean">
  Filter by programs that are currently airing, or not
</ParamField>

<ParamField query="maxStartDate" type="string (DateTime)">
  The maximum premiere start date
</ParamField>

<ParamField query="minEndDate" type="string (DateTime)">
  The minimum premiere end date
</ParamField>

<ParamField query="maxEndDate" type="string (DateTime)">
  The maximum premiere end date
</ParamField>

<ParamField query="isMovie" type="boolean">
  Filter for movies
</ParamField>

<ParamField query="isSeries" type="boolean">
  Filter for series
</ParamField>

<ParamField query="isNews" type="boolean">
  Filter for news
</ParamField>

<ParamField query="isKids" type="boolean">
  Filter for kids content
</ParamField>

<ParamField query="isSports" type="boolean">
  Filter for sports
</ParamField>

<ParamField query="startIndex" type="integer">
  The record index to start at. All items with a lower index will be dropped from the results
</ParamField>

<ParamField query="limit" type="integer">
  The maximum number of records to return
</ParamField>

<ParamField query="sortBy" type="array">
  Sort orders (comma-delimited): Name, StartDate
</ParamField>

<ParamField query="sortOrder" type="array">
  Sort order: Ascending or Descending (comma-delimited)
</ParamField>

<ParamField query="genres" type="array">
  The genres to return guide information for (pipe-delimited)
</ParamField>

<ParamField query="genreIds" type="array">
  The genre IDs to return guide information for (comma-delimited GUIDs)
</ParamField>

<ParamField query="enableImages" type="boolean">
  Include image information in output
</ParamField>

<ParamField query="imageTypeLimit" type="integer">
  The max number of images to return, per image type
</ParamField>

<ParamField query="enableImageTypes" type="array">
  The image types to include in the output (comma-delimited)
</ParamField>

<ParamField query="enableUserData" type="boolean">
  Include user data
</ParamField>

<ParamField query="seriesTimerId" type="string">
  Filter by series timer ID
</ParamField>

<ParamField query="librarySeriesId" type="string (Guid)">
  Filter by library series ID
</ParamField>

<ParamField query="fields" type="array">
  Additional fields of information to return in the output (comma-delimited)
</ParamField>

<ParamField query="enableTotalRecordCount" type="boolean" default="true">
  Retrieve total record count
</ParamField>

### Response

<ResponseField name="Items" type="array">
  Array of program items
  <Expandable title="Program Item Properties">
    <ResponseField name="Id" type="string (Guid)">
      Unique identifier for the program
    </ResponseField>
    <ResponseField name="Name" type="string">
      Program name
    </ResponseField>
    <ResponseField name="ChannelId" type="string">
      Channel ID
    </ResponseField>
    <ResponseField name="StartDate" type="string (DateTime)">
      Program start date and time
    </ResponseField>
    <ResponseField name="EndDate" type="string (DateTime)">
      Program end date and time
    </ResponseField>
    <ResponseField name="Overview" type="string">
      Program description
    </ResponseField>
    <ResponseField name="IsMovie" type="boolean">
      Whether the program is a movie
    </ResponseField>
    <ResponseField name="IsSeries" type="boolean">
      Whether the program is a series
    </ResponseField>
    <ResponseField name="IsSports" type="boolean">
      Whether the program is sports
    </ResponseField>
    <ResponseField name="IsNews" type="boolean">
      Whether the program is news
    </ResponseField>
    <ResponseField name="IsKids" type="boolean">
      Whether the program is for kids
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="TotalRecordCount" type="integer">
  Total number of programs
</ResponseField>

<ResponseField name="StartIndex" type="integer">
  Starting index of the returned results
</ResponseField>

### Status Codes

- `200` - Live TV EPG data returned successfully
- `401` - Unauthorized
- `403` - Forbidden
